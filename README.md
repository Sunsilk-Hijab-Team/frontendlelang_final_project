
## Lelang Online
Aplikasi e-commerce lelang online
# Binar: React.js

## Run Locally

Clone the project

```bash
  git clone https://github.com/Sunsilk-Hijab-Team/frontendlelang_final_project.git
```

Go to the project directory

```bash
  cd frontendlelang_final_project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
